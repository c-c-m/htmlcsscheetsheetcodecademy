Table tags

TAG NAME DESCRIPTION
<table> table The wrapper element for HTML tables
<thead> head The set of rows defining the column headers in a table
<tbody> body The set of rows containing actual table data
<td> table data The table row container
<tr> table row The table row container
<tfoot> table footer The set of rows defining the footer in a table


Table attributes

Atribute Name Discription
colespan Column Span Defines how many columns a td element should span.
rowspan row span Defines how many rows a td element should span.
